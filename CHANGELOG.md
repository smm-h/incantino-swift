# Changelog

## 0.1.0

- Initial release of Incantino iOS SDUI engine
- Expression evaluator with type coercion and operator support
- Scope system for hierarchical data binding
- Flow runner for multi-step navigation sequences
- Formatter pipeline for text transformation
- Popup scheduler with priority and lifecycle management
- Action dispatching system
- Data sources with async loading and caching
- Config providers for runtime configuration
- IncantinoUI SwiftUI components
